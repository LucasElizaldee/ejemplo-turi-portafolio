Robot humanoide Turi

Descripción

En este proyecto se diseñó un robot humanoide inspirado en NAO para apoyar recorridos turísticos dentro de la universidad. El sistema integró ESP32, Arduino UNO, servomotores, ESP32-CAM, audio, aplicación móvil, servidor en Python y comunicación WiFi.

Qué hice en esta actividad

  • Integración de subsistemas electrónicos, mecánicos y de software.
  • Participación en el armado y puesta en funcionamiento del sistema.
  • Pruebas de comunicación entre aplicación, servidor y dispositivos de control.
  • Validación de movimientos, audio, interacción y funcionamiento general.

Competencias desarrolladas

  • Implementar software y hardware específico para el correcto desempeño de procesos.
  • Fabricar equipos, sistemas y procesos mecatrónicos de acuerdo con diseño.
  • Fabricar prototipos para sistemas y procesos mecatrónicos.
  • Colaborar en la integración de nuevas tecnologías realizando pruebas y ensayos.

Reflexión breve

Esta evidencia representa una de las integraciones más completas realizadas, porque combina estructura física, control, comunicación, software e interacción en un mismo sistema. Además, permitió comprobar la importancia de probar cada subsistema por separado antes de integrarlo al conjunto final.
